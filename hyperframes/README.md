# HyperFrames Packaging

This folder is reserved for the optional HyperFrames packaging layer.

The stable renderer remains `scripts/render-demo-video.mjs` with FFmpeg/Pillow. HyperFrames should be used later for agent-generated HTML/CSS packaging such as animated captions, dialogue bubbles, title cards, transitions, stickers, and covers.

Do not commit `node_modules/`, generated videos, or build outputs here.
