# Viral Reference Samples

This folder is optional. It is used for full viral cat meme reference videos that can help analyze pacing, subtitle density, cuts, and story structure.

Current policy:

- Keep small reference videos when they are useful for reproducible demos.
- Do not commit large raw videos if upload is unstable or the file is only an unused original material dump.
- If a reference video is not committed, record its local path, duration, and purpose in this file or in `docs/text-materials.md`.
- The removed `抖音202662-534438.mp4` reference was about 27 seconds long and is treated as unused raw/reference material.

The main reproducible asset library lives in:

- `assets/cat-motions/`
- `assets/backgrounds/`
- `assets/generated/backgrounds/`
- `data/assets-index.json`
